# Shortest Path Finder (CCC Final Project)

## Description
This project is developed in C language to find the shortest path between nodes in a graph. It demonstrates fundamental concepts of graph algorithms and data structures.

## Features
- Calculates shortest path between nodes
- Efficient graph traversal
- Handles weighted graphs

## Technologies Used
- C Programming
- Graph Algorithms

## Concepts Used
- Graph Theory
- Dijkstra / Shortest Path Algorithm
